1. Steamworks SDK 配置步骤

1.1 获取 App ID

在使用 Steamworks 之前，你需要从 Valve 获取一个应用程序 ID。

申请流程

  1. 注册 Steam 开发者账号

    • 访问 Steam 开发者网站
    • 支付一次性费用（约 100 美元）
    • 完成开发者注册流程

  2. 创建应用页面

    • 登录 Steamworks 后台
    • 点击 “Create New App”
    • 填写应用基本信息

  3. 获取 App ID

    • 创建后，系统会分配一个唯一的 App ID（例如：480 是 Spacewar 测试应用）
    • 记下这个 ID，后续配置需要用到

测试用的 App ID

在开发阶段，你可以使用以下测试 App ID：

480 - Spacewar（Valve 提供的测试应用）

使用测试 App ID 时，需要在 Steam 客户端安装 Spacewar 游戏。

1.2 下载和配置 SDK

下载 Steamworks.NET

Steamworks.NET 是 Steamworks SDK 的 C# 封装，适合 Godot C# 使用。

通过 NuGet 安装（推荐）：

# 在项目目录下执行
dotnet add package Steamworks.NET

手动下载：

  1. 访问 Steamworks.NET GitHub
  2. 下载最新 Release
  3. 解压并将以下文件放入项目：
    • Steamworks.NET.dll - 主程序集
    • steam_api.dll / libsteam_api.so / libsteam_api.dylib - 原生库

目录结构

你的项目/
├── 你的项目.csproj
├── steam_appid.txt          # Steam App ID 文件
├── addons/
│   └── steamworks/          # Steamworks 相关脚本
│       └── SteamManager.cs  # 主管理器
└── (平台相关的原生库)
    ├── steam_api64.dll      # Windows 64位
    ├── steam_api.dll        # Windows 32位
    ├── libsteam_api.so      # Linux
    └── libsteam_api.dylib   # macOS

2. Steam 初始化与回调处理

2.1 SteamAPI.Init()

基础初始化代码

using Godot;
using Steamworks;

public partial class SteamInitializer : Node
{
    // Steam 是否成功初始化
    private bool _steamInitialized = false;

    // 当前玩家的 Steam ID
    public CSteamID LocalSteamID { get; private set; }

    // 当前玩家的 Steam 名称
    public string LocalSteamName { get; private set; } = "Player";

    public override void _Ready()
    {
        InitializeSteam();
    }

    /// <summary>
    /// 初始化 Steam 客户端
    /// </summary>
    private void InitializeSteam()
    {
        try
        {
            // 尝试初始化 Steam API
            _steamInitialized = SteamAPI.Init();

            if (_steamInitialized)
            {
                GD.Print("Steam 初始化成功！");

                // 获取本地玩家信息
                LocalSteamID = SteamUser.GetSteamID();
                LocalSteamName = SteamFriends.GetPersonaName();

                GD.Print($"Steam ID: {LocalSteamID}");
                GD.Print($"玩家名称: {LocalSteamName}");

                // 设置游戏信息
                SteamFriends.SetRichPresence("status", "正在游戏中");
                SteamFriends.SetRichPresence("connect", "");
            }
            else
            {
                GD.PushError("Steam 初始化失败！请确保 Steam 客户端已运行");
            }
        }
        catch (System.DllNotFoundException ex)
        {
            GD.PushError($"Steamworks 库未找到: {ex.Message}");
        }
        catch (System.Exception ex)
        {
            GD.PushError($"Steam 初始化异常: {ex.Message}");
        }
    }

    public override void _Process(double delta)
    {
        // 每帧调用 Steam 回调处理
        if (_steamInitialized)
        {
            SteamAPI.RunCallbacks();
        }
    }

    public override void _ExitTree()
    {
        // 游戏退出时关闭 Steam
        if (_steamInitialized)
        {
            SteamAPI.Shutdown();
            GD.Print("Steam 已关闭");
        }
    }
}

2.2 回调系统设置

Steamworks 使用回调机制处理异步事件。你需要在 _Process 中调用 SteamAPI.RunCallbacks()。

回调处理基础

using Godot;
using Steamworks;

public partial class SteamCallbackExample : Node
{
    // 回调委托保护引用，防止被垃圾回收
    private Callback<UserStatsReceived_t> _userStatsReceivedCallback;
    private Callback<GameOverlayActivated_t> _overlayActivatedCallback;
    private CallResult<LeaderboardFindResult_t> _leaderboardFindCallback;

    public override void _Ready()
    {
        InitializeCallbacks();
    }

    /// <summary>
    /// 初始化所有 Steam 回调
    /// </summary>
    private void InitializeCallbacks()
    {
        // 用户统计数据接收回调
        _userStatsReceivedCallback = Callback<UserStatsReceived_t>.Create(
            OnUserStatsReceived);

        // Steam 覆盖层激活回调
        _overlayActivatedCallback = Callback<GameOverlayActivated_t>.Create(
            OnOverlayActivated);

        GD.Print("Steam 回调已注册");
    }

    /// <summary>
    /// 用户统计数据接收处理
    /// </summary>
    private void OnUserStatsReceived(UserStatsReceived_t callback)
    {
        if (callback.m_eResult == EResult.k_EResultOK)
        {
            GD.Print("用户统计数据加载成功");
            // 在这里读取具体统计数据
        }
        else
        {
            GD.PushError($"用户统计数据加载失败: {callback.m_eResult}");
        }
    }

    /// <summary>
    /// Steam 覆盖层激活状态变化
    /// </summary>
    private void OnOverlayActivated(GameOverlayActivated_t callback)
    {
        bool isActive = callback.m_bActive != 0;
        GD.Print($"Steam 覆盖层状态: {(isActive ? "激活" : "关闭")}");

        // 可以在这里暂停游戏
        GetTree().Paused = isActive;
    }

    /// <summary>
    /// 异步调用示例：查找排行榜
    /// </summary>
    public void FindLeaderboard(string leaderboardName)
    {
        SteamAPICall_t handle = SteamUserStats.FindLeaderboard(leaderboardName);
        _leaderboardFindCallback = CallResult<LeaderboardFindResult_t>.Create(
            OnLeaderboardFound);
        _leaderboardFindCallback.Set(handle);
    }

    /// <summary>
    /// 排行榜查找结果回调
    /// </summary>
    private void OnLeaderboardFound(LeaderboardFindResult_t callback, bool ioFailure)
    {
        if (ioFailure)
        {
            GD.PushError("查找排行榜 IO 失败");
            return;
        }

        if (callback.m_bLeaderboardFound != 0)
        {
            GD.Print("排行榜查找成功");
            SteamLeaderboard_t leaderboardId = callback.m_hSteamLeaderboard;
            // 保存 leaderboardId 供后续使用
        }
        else
        {
            GD.PushWarning("排行榜未找到");
        }
    }

    public override void _Process(double delta)
    {
        // 必须每帧调用以处理回调
        SteamAPI.RunCallbacks();
    }
}

2.3 错误处理

完整的错误处理策略

using Godot;
using Steamworks;

public enum SteamInitResult
{
    Success,           // 初始化成功
    SteamNotRunning,   // Steam 未运行
    NoSteamClient,     // 非 Steam 版本
    DllNotFound,       // 库文件缺失
    UnknownError       // 未知错误
}

public partial class SteamErrorHandler : Node
{
    // 当前初始化状态
    public SteamInitResult InitResult { get; private set; } = SteamInitResult.UnknownError;

    // 详细的错误信息
    public string ErrorMessage { get; private set; } = "";

    // Steam 是否可用
    public bool IsSteamAvailable => InitResult == SteamInitResult.Success;

    /// <summary>
    /// 安全初始化 Steam
    /// </summary>
    public SteamInitResult SafeInitialize()
    {
        try
        {
            // 检查 Steam 是否运行（开发测试时可以注释掉）
            if (!IsSteamRunning())
            {
                InitResult = SteamInitResult.SteamNotRunning;
                ErrorMessage = "Steam 客户端未运行，请启动 Steam 后重试";
                GD.PushError(ErrorMessage);
                return InitResult;
            }

            // 尝试初始化
            bool success = SteamAPI.Init();

            if (success)
            {
                InitResult = SteamInitResult.Success;
                ErrorMessage = "";
                GD.Print("Steam 初始化成功");

                // 验证 Steam ID
                CSteamID steamId = SteamUser.GetSteamID();
                if (steamId == CSteamID.Nil)
                {
                    GD.PushWarning("获取到的 Steam ID 无效");
                }
                else
                {
                    GD.Print($"当前玩家: {SteamFriends.GetPersonaName()} ({steamId})");
                }
            }
            else
            {
                InitResult = SteamInitResult.NoSteamClient;
                ErrorMessage = "Steam API 初始化失败，可能不是通过 Steam 启动";
                GD.PushError(ErrorMessage);
            }
        }
        catch (System.DllNotFoundException ex)
        {
            InitResult = SteamInitResult.DllNotFound;
            ErrorMessage = $"Steamworks 库文件缺失: {ex.Message}";
            GD.PushError(ErrorMessage);
        }
        catch (System.BadImageFormatException ex)
        {
            InitResult = SteamInitResult.DllNotFound;
            ErrorMessage = $"Steamworks 库文件格式错误（可能是架构不匹配）: {ex.Message}";
            GD.PushError(ErrorMessage);
        }
        catch (System.Exception ex)
        {
            InitResult = SteamInitResult.UnknownError;
            ErrorMessage = $"Steam 初始化异常: {ex.Message}";
            GD.PushError(ErrorMessage);
        }

        return InitResult;
    }

    /// <summary>
    /// 检查 Steam 客户端是否运行
    /// </summary>
    private bool IsSteamRunning()
    {
        try
        {
            // 通过检查特定进程或文件来判断（简化版本）
            // 实际可以使用更复杂的检测方法
            return true; // 假设总是运行，依赖 SteamAPI.Init 的实际结果
        }
        catch
        {
            return false;
        }
    }

    /// <summary>
    /// 显示错误对话框
    /// </summary>
    public void ShowErrorDialog()
    {
        if (InitResult == SteamInitResult.Success)
            return;

        AcceptDialog dialog = new AcceptDialog();
        dialog.Title = "Steam 错误";
        dialog.DialogText = ErrorMessage;
        dialog.GetLabel().AutowrapMode = TextServer.AutowrapMode.Word;

        AddChild(dialog);
        dialog.PopupCentered();
        dialog.Confirmed += () => dialog.QueueFree();
    }

    public override void _Process(double delta)
    {
        if (IsSteamAvailable)
        {
            SteamAPI.RunCallbacks();
        }
    }

    public override void _ExitTree()
    {
        if (IsSteamAvailable)
        {
            SteamAPI.Shutdown();
        }
    }
}

3. Steam 多人游戏大厅系统

3.1 创建、加入和离开大厅

基础大厅操作

using Godot;
using Steamworks;

public partial class SteamLobbyManager : Node
{
    // 当前大厅 ID
    public CSteamID CurrentLobbyID { get; private set; } = CSteamID.Nil;

    // 是否在大厅中
    public bool InLobby => CurrentLobbyID != CSteamID.Nil;

    // 大厅所有者（主机）
    public CSteamID LobbyOwner => InLobby ? SteamMatchmaking.GetLobbyOwner(CurrentLobbyID) : CSteamID.Nil;

    // 是否是主机
    public bool IsHost => InLobby && LobbyOwner == SteamUser.GetSteamID();

    // 回调委托
    private Callback<LobbyCreated_t> _lobbyCreatedCallback;
    private Callback<LobbyEnter_t> _lobbyEnterCallback;
    private Callback<LobbyMatchList_t> _lobbyMatchListCallback;
    private Callback<LobbyChatUpdate_t> _lobbyChatUpdateCallback;
    private Callback<LobbyDataUpdate_t> _lobbyDataUpdateCallback;
    private Callback<GameLobbyJoinRequested_t> _gameLobbyJoinRequestedCallback;

    [Signal]
    public delegate void LobbyCreatedEventHandler(bool success, ulong lobbyId);

    [Signal]
    public delegate void LobbyJoinedEventHandler(bool success, ulong lobbyId);

    [Signal]
    public delegate void LobbyLeftEventHandler();

    [Signal]
    public delegate void PlayerJoinedLobbyEventHandler(ulong steamId);

    [Signal]
    public delegate void PlayerLeftLobbyEventHandler(ulong steamId);

    public override void _Ready()
    {
        InitializeCallbacks();
    }

    /// <summary>
    /// 初始化 Steam 大厅回调
    /// </summary>
    private void InitializeCallbacks()
    {
        _lobbyCreatedCallback = Callback<LobbyCreated_t>.Create(OnLobbyCreated);
        _lobbyEnterCallback = Callback<LobbyEnter_t>.Create(OnLobbyEntered);
        _lobbyMatchListCallback = Callback<LobbyMatchList_t>.Create(OnLobbyMatchList);
        _lobbyChatUpdateCallback = Callback<LobbyChatUpdate_t>.Create(OnLobbyChatUpdate);
        _lobbyDataUpdateCallback = Callback<LobbyDataUpdate_t>.Create(OnLobbyDataUpdate);
        _gameLobbyJoinRequestedCallback = Callback<GameLobbyJoinRequested_t>.Create(OnGameLobbyJoinRequested);
    }

    // ==================== 大厅创建 ====================

    /// <summary>
    /// 创建一个新的 Steam 大厅
    /// </summary>
    /// <param name="maxPlayers">最大玩家数</param>
    /// <param name="lobbyType">大厅类型</param>
    public void CreateLobby(int maxPlayers = 4, ELobbyType lobbyType = ELobbyType.k_ELobbyTypePublic)
    {
        if (InLobby)
        {
            GD.PushWarning("已经在 lobby 中，先离开当前 lobby");
            return;
        }

        GD.Print($"正在创建 lobby，最大玩家数: {maxPlayers}...");
        SteamAPICall_t result = SteamMatchmaking.CreateLobby(lobbyType, maxPlayers);

        if (result == SteamAPICall_t.Invalid)
        {
            GD.PushError("创建 lobby 请求失败");
        }
    }

    /// <summary>
    /// 创建私密 Lobby（仅限邀请）
    /// </summary>
    public void CreatePrivateLobby(int maxPlayers = 4)
    {
        CreateLobby(maxPlayers, ELobbyType.k_ELobbyTypePrivate);
    }

    /// <summary>
    /// 创建好友可见 Lobby
    /// </summary>
    public void CreateFriendsOnlyLobby(int maxPlayers = 4)
    {
        CreateLobby(maxPlayers, ELobbyType.k_ELobbyTypeFriendsOnly);
    }

    /// <summary>
    /// Lobby 创建完成回调
    /// </summary>
    private void OnLobbyCreated(LobbyCreated_t callback)
    {
        if (callback.m_eResult == EResult.k_EResultOK)
        {
            CurrentLobbyID = new CSteamID(callback.m_ulSteamIDLobby);
            GD.Print($"Lobby 创建成功: {CurrentLobbyID}");

            // 设置 Lobby 数据
            SetupLobbyData();

            EmitSignal(SignalName.LobbyCreated, true, (ulong)CurrentLobbyID);
        }
        else
        {
            GD.PushError($"Lobby 创建失败: {callback.m_eResult}");
            EmitSignal(SignalName.LobbyCreated, false, (ulong)0);
        }
    }

    /// <summary>
    /// 设置 Lobby 基础数据
    /// </summary>
    private void SetupLobbyData()
    {
        if (!InLobby) return;

        // 设置 Lobby 名称
        string lobbyName = $"{SteamFriends.GetPersonaName()} 的游戏";
        SteamMatchmaking.SetLobbyData(CurrentLobbyID, "name", lobbyName);

        // 设置游戏模式
        SteamMatchmaking.SetLobbyData(CurrentLobbyID, "mode", "casual");

        // 设置地图
        SteamMatchmaking.SetLobbyData(CurrentLobbyID, "map", "default");

        // 设置版本号（用于版本匹配）
        SteamMatchmaking.SetLobbyData(CurrentLobbyID, "version", "1.0.0");

        // 设置游戏标识
        SteamMatchmaking.SetLobbyData(CurrentLobbyID, "game", "YourGameName");
    }

    // ==================== 加入 Lobby ====================

    /// <summary>
    /// 通过 Lobby ID 加入
    /// </summary>
    public void JoinLobby(ulong lobbyId)
    {
        if (InLobby)
        {
            GD.PushWarning("已在 lobby 中，先离开当前 lobby");
            return;
        }

        CSteamID id = new CSteamID(lobbyId);
        GD.Print($"正在加入 lobby: {lobbyId}...");
        SteamMatchmaking.JoinLobby(id);
    }

    /// <summary>
    /// 加入 Lobby 结果回调
    /// </summary>
    private void OnLobbyEntered(LobbyEnter_t callback)
    {
        if (callback.m_EChatRoomEnterResponse == (uint)EChatRoomEnterResponse.k_EChatRoomEnterResponseSuccess)
        {
            CurrentLobbyID = new CSteamID(callback.m_ulSteamIDLobby);
            GD.Print($"成功加入 lobby: {CurrentLobbyID}");
            GD.Print($"Lobby 成员数: {SteamMatchmaking.GetNumLobbyMembers(CurrentLobbyID)}");

            EmitSignal(SignalName.LobbyJoined, true, (ulong)CurrentLobbyID);
        }
        else
        {
            GD.PushError($"加入 lobby 失败: {callback.m_EChatRoomEnterResponse}");
            EmitSignal(SignalName.LobbyJoined, false, (ulong)0);
        }
    }

    /// <summary>
    /// 通过好友邀请加入
    /// </summary>
    private void OnGameLobbyJoinRequested(GameLobbyJoinRequested_t callback)
    {
        GD.Print("收到 lobby 邀请，正在加入...");
        JoinLobby(callback.m_steamIDLobby.m_SteamID);
    }

    // ==================== 离开 Lobby ====================

    /// <summary>
    /// 离开当前 Lobby
    /// </summary>
    public void LeaveLobby()
    {
        if (!InLobby) return;

        GD.Print($"离开 lobby: {CurrentLobbyID}");
        SteamMatchmaking.LeaveLobby(CurrentLobbyID);
        CurrentLobbyID = CSteamID.Nil;

        EmitSignal(SignalName.LobbyLeft);
    }

    /// <summary>
    /// Lobby 成员变动回调
    /// </summary>
    private void OnLobbyChatUpdate(LobbyChatUpdate_t callback)
    {
        CSteamID changedUser = new CSteamID(callback.m_ulSteamIDUserChanged);
        EChatMemberStateChange stateChange = (EChatMemberStateChange)callback.m_rgfChatMemberStateChange;

        if (stateChange == EChatMemberStateChange.k_EChatMemberStateChangeEntered)
        {
            GD.Print($"玩家加入: {SteamFriends.GetFriendPersonaName(changedUser)}");
            EmitSignal(SignalName.PlayerJoinedLobby, (ulong)changedUser);
        }
        else if (stateChange == EChatMemberStateChange.k_EChatMemberStateChangeLeft ||
                 stateChange == EChatMemberStateChange.k_EChatMemberStateChangeDisconnected)
        {
            GD.Print($"玩家离开: {SteamFriends.GetFriendPersonaName(changedUser)}");
            EmitSignal(SignalName.PlayerLeftLobby, (ulong)changedUser);
        }
    }

    /// <summary>
    /// Lobby 数据更新回调
    /// </summary>
    private void OnLobbyDataUpdate(LobbyDataUpdate_t callback)
    {
        CSteamID lobbyId = new CSteamID(callback.m_ulSteamIDLobby);
        CSteamID memberId = new CSteamID(callback.m_ulSteamIDMember);

        GD.Print($"Lobby 数据更新: Lobby={lobbyId}, Member={memberId}");
    }

    public override void _Process(double delta)
    {
        SteamAPI.RunCallbacks();
    }

    public override void _ExitTree()
    {
        LeaveLobby();
    }
}

using Godot;
using Steamworks;
using System.Collections.Generic;

public partial class SteamLobbyBrowser : Node
{
    // 搜索到的 Lobby 列表
    public List<CSteamID> FoundLobbies { get; private set; } = new List<CSteamID>();

    // 是否正在搜索
    public bool IsSearching { get; private set; } = false;

    // 回调
    private Callback<LobbyMatchList_t> _lobbyMatchListCallback;

    [Signal]
    public delegate void SearchCompletedEventHandler(int lobbyCount);

    public override void _Ready()
    {
        _lobbyMatchListCallback = Callback<LobbyMatchList_t>.Create(OnLobbyMatchList);
    }

    // ==================== 基础搜索 ====================

    /// <summary>
    /// 搜索所有 Lobby
    /// </summary>
    public void SearchAllLobbies()
    {
        if (IsSearching) return;

        IsSearching = true;
        FoundLobbies.Clear();

        GD.Print("开始搜索 lobby...");
        SteamAPICall_t handle = SteamMatchmaking.RequestLobbyList();
    }

    /// <summary>
    /// 带距离过滤的搜索
    /// </summary>
    public void SearchNearbyLobbies(ELobbyDistanceFilter distance = ELobbyDistanceFilter.k_ELobbyDistanceFilterDefault)
    {
        if (IsSearching) return;

        IsSearching = true;
        FoundLobbies.Clear();

        // 设置搜索距离

    {
        CSteamID senderId = callback.m_steamIDRemote;

        EmitSignal(SignalName.P2PConnectionRequest, (ulong)senderId);

    private void OnP2PSessionConnectFail(P2PSessionConnectFail_t callback)

    {
        if (data.Length > MaxPacketSize)

            GD.PushError($"数据包过大: {data.Length} > {MaxPacketSize}");
            return false;

        bool success = SteamNetworking.SendP2PPacket(
            targetId,

            (uint)data.Length,
            EP2PSend.k_EP2PSendUnreliableNoDelay,
            channel);

            GD.PushError($"发送 P2P 数据包失败到: {targetId}");

        bool success = SteamNetworking.SendP2PPacket(
            targetId,
            data,
            (uint)data.Length,
            EP2PSend.k_EP2PSendReliable,
            channel);

        return success;
    }

        foreach (var session in _sessions.Keys)

        }
    }

            {
                EmitSignal(SignalName.P2PDataReceived, (ulong)senderId, data);
            }
        }
    }

    {
        SteamAPI.RunCallbacks();
        ProcessP2PPackets();
    }

        foreach (var session in _sessions.Keys)
        {
            SteamNetworking.CloseP2PSessionWithUser(session);
        }
        _sessions.Clear();
    }

using Godot;
using Steamworks;
using System;

/// <summary>
/// P2P 网络通道管理器
/// 支持多个独立通道，分别用于不同类型数据
/// </summary>
public partial class SteamP2PChannels : Node
{
    // 通道定义
    public const int ChannelReliable = 0;       // 可靠消息（重要游戏事件）
    public const int ChannelUnreliable = 1;     // 不可靠消息（位置更新等）
    public const int ChannelVoice = 2;          // 语音数据
    public const int ChannelSystem = 3;         // 系统消息
    public const int ChannelChat = 4;           // 聊天消息

    // 统计信息
    private P2PChannelStats[] _channelStats = new P2PChannelStats[5];

    public override void _Ready()
    {
        // 初始化各通道统计
        for (int i = 0; i < _channelStats.Length; i++)
        {
            _channelStats[i] = new P2PChannelStats { ChannelId = i };
        }
    }

    /// <summary>
    /// 使用指定通道发送数据
    /// </summary>
    public bool SendOnChannel(CSteamID target, byte[] data, int channel, bool forceReliable = false)
    {
        EP2PSend sendType;

        if (forceReliable || channel == ChannelReliable || channel == ChannelSystem)
        {
            sendType = EP2PSend.k_EP2PSendReliable;
        }
        else if (channel == ChannelUnreliable)
        {
            sendType = EP2PSend.k_EP2PSendUnreliableNoDelay;
        }
        else if (channel == ChannelVoice)
        {
            sendType = EP2PSend.k_EP2PSendUnreliable;
        }
        else
        {
            sendType = EP2PSend.k_EP2PSendReliableWithBuffering;
        }

        bool success = SteamNetworking.SendP2PPacket(target, data, (uint)data.Length, sendType, channel);

        if (success)
        {
            _channelStats[channel].PacketsSent++;
            _channelStats[channel].BytesSent += data.Length;
        }
        else
        {
            _channelStats[channel].PacketsFailed++;
        }

        return success;
    }

    /// <summary>
    /// 发送可靠的游戏事件
    /// </summary>
    public bool SendGameEvent(CSteamID target, byte[] data)
    {
        return SendOnChannel(target, data, ChannelReliable, true);
    }

    /// <summary>
    /// 发送位置更新（不可靠，快速）
    /// </summary>
    public bool SendPositionUpdate(CSteamID target, byte[] data)
    {
        return SendOnChannel(target, data, ChannelUnreliable);
    }

    /// <summary>
    /// 发送聊天消息
    /// </summary>
    public bool SendChatMessage(CSteamID target, string message)
    {
        byte[] data = System.Text.Encoding.UTF8.GetBytes(message);
        return SendOnChannel(target, data, ChannelChat, true);
    }

    /// <summary>
    /// 发送系统消息
    /// </summary>
    public bool SendSystemMessage(CSteamID target, byte[] data)
    {
        return SendOnChannel(target, data, ChannelSystem, true);
    }

    /// <summary>
    /// 读取指定通道的数据包
    /// </summary>
    public bool ReadFromChannel(out byte[] data, out CSteamID sender, int channel)
    {
        uint size;
        if (SteamNetworking.IsP2PPacketAvailable(out size, channel))
        {
            data = new byte[size];
            if (SteamNetworking.ReadP2PPacket(data, size, out _, out sender, channel))
            {
                _channelStats[channel].PacketsReceived++;
                _channelStats[channel].BytesReceived += (int)size;
                return true;
            }
        }

        data = null;
        sender = CSteamID.Nil;
        return false;
    }

    /// <summary>
    /// 获取通道统计
    /// </summary>
    public P2PChannelStats GetChannelStats(int channel)
    {
        if (channel >= 0 && channel < _channelStats.Length)
        {
            return _channelStats[channel];
        }
        return null;
    }

    /// <summary>
    /// 打印所有通道统计
    /// </summary>
    public void PrintChannelStats()
    {
        GD.Print("=== P2P 通道统计 ===");
        foreach (var stats in _channelStats)
        {
            GD.Print($"通道 {stats.ChannelId}: " +
                     $"发送 {stats.PacketsSent} 包 ({FormatBytes(stats.BytesSent)}), " +
                     $"接收 {stats.PacketsReceived} 包 ({FormatBytes(stats.BytesReceived)}), " +
                     $"失败 {stats.PacketsFailed}");
        }
    }

    private string FormatBytes(long bytes)
    {
        if (bytes < 1024) return $"{bytes} B";
        if (bytes < 1024 * 1024) return $"{bytes / 1024.0:F2} KB";
        return $"{bytes / (1024.0 * 1024):F2} MB";
    }
}

        /// <summary>
        /// 接受来自 Lobby 成员的 P2P 连接
        /// </summary>
        private void AcceptP2PFromLobbyMembers()
        {
            if (!InLobby) return;

            var members = GetLobbyMembers();

            {
                if (memberId != (ulong)LocalSteamID)

            }

        /// <summary>
        /// P2P 连接请求回调
        /// </summary>
        private void OnP2PSessionRequest(P2PSessionRequest_t callback)
        {
            CSteamID senderId = callback.m_steamIDRemote;

            // 检查是否是 Lobby 成员
            if (InLobby && GetLobbyMembers().Contains((ulong)senderId))
            {
                SteamNetworking.AcceptP2PSessionWithUser(senderId);

                _p2pSessions[senderId] = new P2PSessionInfo
                {
                    SteamId = senderId,
                    ConnectedTime = DateTime.Now,
                    IsConnected = true
                };

                string playerName = SteamFriends.GetFriendPersonaName(senderId);
                GD.Print($"P2P 连接建立: {playerName}");

                EmitSignal(SignalName.P2PConnected, (ulong)senderId);
            }
        }

        /// P2P 连接失败回调
        /// </summary>
        private void OnP2PSessionConnectFail(P2PSessionConnectFail_t callback)
        {
            CSteamID failedId = callback.m_steamIDRemote;
            EP2PSessionError error = (EP2PSessionError)callback.m_eP2PSessionError;

            GD.PushError($"P2P 连接失败: {failedId}, 错误: {error}");

            _p2pSessions.Remove(failedId);
            EmitSignal(SignalName.P2PDisconnected, (ulong)failedId);
        }

        /// <summary>
        /// 发送 P2P 数据包
        /// </summary>
        public bool SendP2PData(ulong targetSteamId, byte[] data, bool reliable = true)
        {
            if (!IsInitialized) return false;

            CSteamID target = new CSteamID(targetSteamId);
            EP2PSend sendType = reliable ? EP2PSend.k_EP2PSendReliable : EP2PSend.k_EP2PSendUnreliableNoDelay;

            return SteamNetworking.SendP2PPacket(target, data, (uint)data.Length, sendType);
        }

        /// <summary>
        /// 广播 P2P 数据给所有连接的玩家
        /// </summary>
        public void BroadcastP2PData(byte[] data, bool reliable = true)
        {
            if (!IsInitialized) return;

            EP2PSend sendType = reliable ? EP2PSend.k_EP2PSendReliable : EP2PSend.k_EP2PSendUnreliableNoDelay;

            foreach (var session in _p2pSessions.Keys.ToList())
            {
                SteamNetworking.SendP2PPacket(session, data, (uint)data.Length, sendType);
            }
        }

        /// <summary>
        /// 处理 P2P 数据包
        /// </summary>
        private void ProcessP2PPackets()
        {
            uint packetSize;

            while (SteamNetworking.IsP2PPacketAvailable(out packetSize))
            {
                byte[] data = new byte[packetSize];
                CSteamID senderId;

                if (SteamNetworking.ReadP2PPacket(data, packetSize, out _, out senderId))
                {
                    _packetQueue.Enqueue(new P2PPacket
                    {
                        SenderId = senderId,
                        Data = data,
                        ReceiveTime = DateTime.Now
                    });

                    EmitSignal(SignalName.P2PDataReceived, (ulong)senderId, data);
                }
            }
        }

14.7 核心概念深入讲解

14.7.1 Steam 平台架构解析

Steam平台不仅仅是一个游戏商店，它是一个完整的游戏生态系统。理解其架构有助于更好地利用Steamworks API：

Steam 客户端层：运行在玩家电脑上的Steam客户端负责用户认证、好友列表、云存档同步等核心功能。游戏通过Steam API与客户端通信，而非直接连接到Steam服务器。这种设计减轻了服务器的负载，同时提供了更好的隐私保护。

Steam 服务器层：Valve运营着分布在全球的服务器集群，处理匹配、排行榜、成就同步等服务。这些服务器使用CDN加速，确保全球玩家都能获得低延迟的体验。

P2P 中继网络：Steam的P2P连接通常不直接建立，而是通过Steam的中继服务器转发。这种设计解决了NAT穿透问题，同时提供了DDoS保护。开发者无需担心复杂的网络配置。

14.7.2 Steamworks 回调机制

Steamworks使用C++风格的回调系统，在C#中需要特别注意：

回调的生命周期：回调对象必须保持引用，否则会被垃圾回收导致回调失效。通常将回调声明为类的成员变量：

private Callback<UserStatsReceived_t> _userStatsCallback;

回调的执行时机：SteamAPI.RunCallbacks()必须在主线程每帧调用。在Godot中，通常在_Process方法中调用。遗漏调用会导致事件积压和延迟响应。

CallResult 与 Callback 的区别：

  • Callback：用于持续监听的事件（如覆盖层打开、好友状态变化）
  • CallResult：用于一次性异步操作的结果（如查找排行榜、上传分数）

14.7.3 Steam 大厅的设计理念

Steam的大厅系统（Lobby）是一个轻量级的房间管理系统：

大厅 vs 游戏服务器：大厅不运行游戏逻辑，它只是玩家聚集的"会议室"。实际的游戏连接通过P2P或独立的游戏服务器建立。这种分离让大厅非常轻量，一个大厅实例只需要很少的Steam服务器资源。

大厅元数据：大厅数据存储在Steam服务器上，所有成员都可以读取，但只有所有者可以写入。这种设计适合存储游戏模式、地图、版本等配置信息，但不适合频繁变化的数据（如玩家位置）。

大厅搜索：Steam支持基于元数据的搜索过滤，但搜索是在Steam服务器端执行的。过多的搜索请求可能触发速率限制，应该缓存搜索结果。

14.8 实现细节深度分析

14.8.1 Steam P2P 网络的底层机制

Steam的P2P网络比直接使用UDP更复杂但更可靠：

NAT 穿透策略：

  1. 直接连接（如果双方有公网IP）
  2. NAT打洞（通过STUN服务器获取公网地址）
  3. TURN中继（通过Steam服务器转发）

Steam自动选择最佳路径，开发者无需关心实现细节。

通道管理： Steam P2P支持最多4个独立通道，每个通道有自己的可靠性设置：

  • 通道0：可靠有序（游戏事件）
  • 通道1：不可靠（位置更新）
  • 通道2：可靠无序（聊天消息）
  • 通道3：保留

数据包大小限制： Steam P2P建议数据包不超过1200字节，这是为了避免IP分片。如果数据更大，需要应用层分片。

14.8.2 成就与统计的同步策略

成就和统计系统看似简单，但背后有复杂的同步逻辑：

本地缓存：成就和统计首先写入本地缓存，然后异步上传到Steam服务器。这确保了即使网络不可用，游戏也能正常运行。

冲突解决：如果玩家在不同设备上玩游戏，可能出现统计冲突。Steam使用"最后写入 wins"策略，但这可能导致数据丢失。对于重要统计，建议游戏自己实现合并逻辑。

成就解锁的时机：

  • 立即解锁：适合教程成就、一次性事件
  • 关卡结束时解锁：适合通关成就
  • 延迟解锁：适合连续登录等需要验证的成就

统计的批量更新：频繁的统计更新（如每帧更新击杀数）应该批量处理，避免过多的网络请求。

14.8.3 云存档的冲突处理

云存档在不同设备间同步时可能产生冲突：

冲突检测：Steam通过文件时间戳检测冲突。如果本地文件比云端新，会上传；如果云端更新，会下载。

自动合并：Steam不提供自动合并功能，游戏需要自己实现。策略包括：

  • 最新存档优先（简单但可能丢失进度）
  • 合并存档（复杂但数据完整）
  • 让玩家选择（最安全但体验差）

存档格式建议：

  • 使用JSON或二进制格式
  • 包含版本号和时间戳
  • 支持部分更新（只保存变化的部分）

存档大小限制： Steam云存档通常有100MB-1GB的限制（取决于Steam配置）。超过限制会导致同步失败。

14.9.1 Steam 集成开发流程

发布后：

  1. 监控Steam错误日志
  2. 关注玩家反馈的网络问题
  3. 定期更新成就和DLC支持

14.9.2 常见错误处理

SteamAPI_Init() 失败：

  • 检查steam_appid.txt是否存在且内容正确
  • 确保Steam客户端正在运行且已登录
  • 检查Steamworks库文件是否正确部署
  • 确认Steam用户拥有该游戏（发布前使用480测试）

回调不触发：

  • 确保每帧调用SteamAPI.RunCallbacks()
  • 检查回调对象是否被垃圾回收
  • 验证回调注册代码是否执行
  • 确认Steam客户端没有离线

P2P 连接失败：

  • 检查防火墙设置
  • 确认双方都在同一个大厅
  • 验证是否调用了AcceptP2PSessionWithUser
  • 检查NAT类型（严格NAT可能导致问题）

成就解锁失败：

  • 确保统计数据已加载（RequestCurrentStats成功）
  • 检查成就ID是否与Steam后台配置一致
  • 验证SetAchievement返回true
  • 确认调用了StoreStats()

14.9.3 性能与用户体验优化

启动优化：

  • 延迟初始化非关键Steam功能（如好友列表）
  • 使用异步API避免阻塞主线程
  • 如果Steam初始化失败，提供离线模式选项

运行时优化：

  • 批量处理统计更新（不要每帧更新）
  • 限制云存档的写入频率（使用防抖）
  • 在非关键帧处理Steam回调（如加载屏幕）

用户体验：

  • 显示Steam成就解锁的自定义UI（而非仅依赖Steam通知）
  • 提供云存档同步状态的视觉反馈
  • 在设置中允许玩家禁用某些Steam功能（如覆盖层）
  • 处理Steam覆盖层激活时的游戏暂停

网络优化：

  • 使用Steam的P2P连接质量API选择最佳主机
  • 实现断线重连机制
  • 当Steam连接不可用时回退到直连模式
  • 监控网络质量并动态调整同步频率

14.9.4 安全考虑

防篡改： Steam的成就和统计可以在客户端被修改。重要数据应该在服务器端验证，Steam统计仅用于展示。

反作弊： Steam提供VAC（Valve Anti-Cheat）集成，但需要额外申请。对于竞技游戏，建议使用专业的反作弊服务。

隐私合规：

  • 在使用Steam好友数据前获取玩家同意
  • 遵守GDPR等数据保护法规
  • 提供隐私政策说明数据使用方式
  • 允许玩家导出和删除自己的数据
